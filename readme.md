## Install

```bash
# install npm dependencies
npm install
# install wasm-pack
curl https://rustwasm.github.io/wasm-pack/installer/init.sh -sSf | sh
```

## Build

```bash
# check wasm-pack installation
wasm-pack -V

# build with webpack
npm run build
```

# Serve

```bash
# start the dev server and compiler
npm start
```
